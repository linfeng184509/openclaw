# OpenClaw 完全指南

> 构建你的个人 AI 助手网关

## 关于本书

本书是 OpenClaw 项目的完整指南，面向所有类型的读者：

- **初学者**：从零开始学习 OpenClaw 的安装和配置
- **开发者**：深入了解架构设计、插件开发和 API 集成
- **运维人员**：掌握生产环境部署、监控和故障排除

## 书籍结构

| 部分 | 内容 | 适合读者 |
|------|------|----------|
| 第一部分：入门篇 | 基础概念、安装配置 | 初学者 |
| 第二部分：配置篇 | 网关、通道、模型配置 | 所有读者 |
| 第三部分：进阶篇 | 多代理、工具、技能开发 | 开发者 |
| 第四部分：平台篇 | macOS、iOS、Android 应用 | 所有读者 |
| 第五部分：运维篇 | 部署、自动化、故障排除 | 运维人员 |
| 第六部分：参考篇 | CLI、配置、API 参考 | 所有读者 |

## 阅读建议

### 如果你是初学者
1. 从第 1 章开始，了解 OpenClaw 是什么
2. 按顺序阅读第 2-3 章，完成基础配置
3. 根据需要阅读配置篇的相关章节

### 如果你是开发者
1. 快速浏览第 1-3 章了解基础
2. 重点阅读第 8-11 章的进阶内容
3. 参考附录中的 API 文档

### 如果你是运维人员
1. 了解第 1-3 章的基础概念
2. 重点阅读第 16-18 章的运维内容
3. 参考附录中的配置 Schema
